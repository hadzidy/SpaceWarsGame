# SpaceWarsGame

    Spacial shooter
    Game Prototype.

TypeScript Powered.

    -OOP constructs to JavaScript.
    -JavaScript superset that generates plain JavaScript.

#Ussage

 Requirements

    -Gulp
    -Node
    -Bower


    1. clone the repo

       ~ git clone https://github.com/hadzidy/SpaceWarsGame.git

    2. install npm

       ~ npm install

    3. installing bower dependencies

        ~ bower install

    4. run default task

        ~ gulp

    *productions task

        ~ gulp build
